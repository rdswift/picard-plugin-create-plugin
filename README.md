# Create Local Plugin

This plugin assists in the creation of a local plugin by preparing the plugin framework files and git repository. It gathers some information from the user such as author, title, description, license and target directory, and then writes the basic framework files to the specified directory and initializes the git repository.

Please see the [User Guide](https://picard-plugins-user-guides.readthedocs.io/en/latest/create_plugin/user_guide.html) for more information.
